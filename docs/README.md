# Red Ride Ticket Source Material

This folder archives the strategic source material behind Red Ride Ticket. These documents support the public site, the venture studio positioning, the sales motion, and the portfolio case study.

## Documents

- `source-material/Venture-Execution-Studio.pdf`
  - Public-facing venture execution narrative.
  - Core message: stop financing the void; start constructing the asset.
  - Frames the Venture Execution Engine and the bridge from technical friction to invoicing.

- `source-material/Venture-Structural-Integrity.pdf`
  - Structural validation narrative.
  - Frames the diagnosis model around problem hypothesis, micro-friction analysis, and high-pain gap identification.
  - Supports the entry offer and the studio's "construction over assembly" philosophy.

- `source-material/Pricing-Integrity-Venture-Execution.pdf`
  - Price integrity deck for venture execution.
  - Core message: Red Ride Ticket does not sell development hours; it sells certainty and architectural logic.

- `source-material/Pricing-Integrity-Venture-Studio.pdf`
  - Price integrity deck for the venture studio framing.
  - Reinforces selective access, market positioning, and pricing discipline.

- `source-material/RED_RIDE_TICKET_SALES_PLAYBOOK_V01.docx`
  - Internal sales playbook.
  - Defines the ICP, pain model, sales conversation, objections, and close.
  - Establishes the Structural Validation offer at USD 2,500.

- `source-material/Red_Ride_Ticket_Infrastructure_Proposal_V2.pdf`
  - Internal infrastructure proposal.
  - Evaluates AI infrastructure consolidation, asset generation, motion generation, coding support, and storage/data capacity.
